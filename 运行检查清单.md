# LLDP Network Analyzer v2.0 - 运行检查清单

## 🚀 启动前检查

- [ ] **Npcap驱动已安装**
  - 检查方法: 在"程序和功能"中查看是否有Npcap
  - 如未安装: 从 https://npcap.com/ 下载安装
  - ⚠️ **必须**: 安装时选择"支持Raw 802.11流量"选项

- [ ] **管理员权限准备**
  - 右键点击 `启动LLDP分析器.bat`
  - 选择"以管理员身份运行"

- [ ] **网络连接准备**
  - 确保至少有一个物理网卡已启用
  - 准备好连接到目标网络设备的网线

## ✅ 启动测试

- [ ] **应用程序启动**
  - 运行 `启动LLDP分析器.bat` 或 `dist\LLDP_Analyzer_v2.exe`
  - 应该看到主窗口界面，无错误提示
  - 检查控制台是否有错误信息

- [ ] **网卡自动选择**
  - 应用启动后应自动选择最佳网卡
  - 下拉菜单中应显示可用的网络适配器
  - 状态栏应显示找到的网卡数量

- [ ] **界面显示正常**
  - 所有标签和按钮显示正确
  - 中文文本无乱码
  - 窗口大小和布局正常

## 🔍 功能测试

- [ ] **网络扫描功能**
  - 点击"刷新网卡列表"按钮
  - 能否正确扫描和显示网络适配器
  - 网卡描述信息是否完整

- [ ] **设备发现功能**
  - 选择正确的网卡
  - 点击"开始捕获"
  - 将网线插入到目标设备
  - 应能在60秒内自动发现设备

- [ ] **设备信息显示**
  - 设备名称、型号、MAC地址显示正确
  - 端口信息、VLAN信息显示完整
  - 技术信息（速率、双工、PoE等）显示正确

- [ ] **网络意图分析**
  - 端口角色推断显示（Access/Trunk/Uplink等）
  - 置信度评分合理
  - 运维洞察和配置建议显示

## 📊 数据导出测试

- [ ] **JSON导出**
  - 点击"导出"按钮
  - 选择JSON格式
  - 检查导出的JSON文件格式正确

- [ ] **CSV导出**
  - 选择CSV格式
  - 检查CSV文件能用Excel打开
  - 数据完整性验证

- [ ] **TXT导出**
  - 选择TXT格式
  - 检查文本格式可读性
  - 信息完整性验证

## 🔧 故障排除

### 如果启动失败:
1. 检查是否以管理员身份运行
2. 重新安装Npcap驱动
3. 检查防火墙设置
4. 查看Windows事件查看器中的错误信息

### 如果无法发现设备:
1. 确认网线已正确连接
2. 检查目标设备是否启用LLDP/CDP
3. 尝试不同的网卡
4. 检查是否有VLAN隔离

### 如果显示异常:
1. 重启应用程序
2. 清除临时文件
3. 更新显卡驱动
4. 检查系统DPI设置

- ✅ 设备信息显示完整准确
- ✅ 数据导出功能正常工作
- ✅ 性能表现良好，无明显卡顿
- ✅ 错误处理友好，提示清晰
