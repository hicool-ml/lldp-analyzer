### **4. ✅ try_decode_text工具函数**

#### **改进前**
```python
# ❌ 旧代码：重复的解码逻辑
# H3C TLV
for encoding in ['utf-8', 'gbk', 'ascii', 'latin-1']:
    try:
        decoded = payload.decode(encoding, errors='strict').strip()
        if decoded and len(decoded) > 0:
            if any(c.isprintable() and not c.isspace() for c in decoded):
                string_value = decoded
                break
    except (UnicodeDecodeError, UnicodeError):
        continue

# Vendor TLV (同样逻辑重复)
# VLAN name (同样逻辑重复)
```

#### **改进后**
```python
# ✅ 新代码：统一工具函数
def try_decode_text(payload: bytes, encodings: List[str] = None) -> Optional[str]:
    """🔥 工具函数：尝试多种编码方式解码文本"""
    if encodings is None:
        encodings = ['utf-8', 'gbk', 'ascii', 'latin-1']

    for encoding in encodings:
        try:
            decoded = payload.decode(encoding)
            cleaned = ''.join(char if char.isprintable() else ' ' for char in decoded)
            return cleaned.strip()
        except (UnicodeDecodeError, LookupError):
            continue

    return payload[:MAX_HEX_DISPLAY].hex() if len(payload) > MAX_HEX_DISPLAY else payload.hex()

# 使用示例（简化版）
string_value = try_decode_text(vlan_name_bytes)
```

#### **优势**
- ✅ **代码复用**: 4处重复逻辑统一为1个函数
- ✅ **维护性**: 解码逻辑集中管理
- ✅ **测试性**: 工具函数可独立测试
- ✅ **一致性**: 所有解码行为一致
