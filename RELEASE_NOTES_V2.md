## 📊 端口角色推断

### Terminal (橙色 - 置信度85%)
**特征**: PoE + 百兆/千兆 + 单VLAN
**用途**: AP/Phone/IP Camera接入

### Trunk (蓝色 - 置信度90%)
**特征**: Port VLAN + Protocol VLAN + 高速
**用途**: 设备间干道互联

### Access (绿色 - 置信度70%)
**特征**: 单一VLAN + 无PoE
**用途**: 普通接入端口

### Uplink (浅紫 - 置信度80%)
**特征**: 路由能力 + 高速
**用途**: 网络上联

### Uplink LAG (深紫 - 置信度95%)
**特征**: 链路聚合 + 高速
**用途**: 核心聚合上联
